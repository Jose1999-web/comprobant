
🚀 Bot de comprobantes para Telegram (24/7 en Render.com)

1. Crea un repositorio en GitHub y sube el contenido de esta carpeta.
2. Ve a https://render.com y crea una cuenta (puedes usar Google).
3. Haz clic en "New" > "Web Service" y conecta tu GitHub.
4. Render detectará el archivo render.yaml automáticamente.
5. En "TELEGRAM_TOKEN", pega el token real de BotFather (no lo pongas en el código).
6. Dale Deploy y ¡listo! Tu bot estará activo 24/7.

Usa el comando /comprobante desde Telegram.
